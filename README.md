# Tic Tac Toe Bot

A simple Tic Tac Toe game made with HTML, CSS, and plain JavaScript.

## Features
- Play against a bot
- 3 difficulty levels: Easy, Medium, Hard
- Light / Dark mode
- Simple animations
- Runs fully in the browser

## Files
- `index.html` – main page  
- `style.css` – styles  
- `script.js` – game logic
